# Organizador Automático de Archivos
Este script en Python organiza automáticamente los archivos de mi carpeta de descargas en categorías (Documentos, Imágenes, Otros).

## Cómo usarlo
1. Instala Python.
2. Cambia la variable `carpeta_descargas` en el script por tu propia ruta.
3. Ejecuta con: `python nombre_del_archivo.py`
